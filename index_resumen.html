Resumen sobre AJAX

AJAX es un patrón de diseño para crear aplicaciones web que interactúan con el servidor sin recargar la página completa. Esto permite actualizar partes de la interfaz sin necesidad de recargar la totalidad de la página.

Algunas de las ventajas de AJAX son:

  • Actualización parcial de la página
  • Reducción de tiempo de carga
  • Mejoras en la experiencia de usuario
Para usar AJAX en HTML, se puede utilizar la etiqueta <script> con el atributo src para cargar el código JavaScript que implementa AJAX.

Para realizar solicitudes AJAX, se puede utilizar el objeto XMLHttpRequest o el objeto fetch (nativo en los navegadores modernos).

Un ejemplo de cómo usar AJAX para cargar información de un usuario:

    
    const userId = 1; // ID del usuario a cargar
    
    fetch(`https://jsonplaceholder.typicode.com/users/${userId}`)
     .then(response => response.json())
     .then(user => {
        document.getElementById('userInfo').innerHTML = `
        Nombre: ${user.name}
        Email: ${user.email}
        Teléfono: ${user.phone}
        `;
    });
